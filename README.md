# NeoFastTaxCalc

NeoFastTaxCalc is a program which purpose is to count the yearly Personal Income Tax(PIT) or Corporate Income Tax(CIT) in Polish tax system based on the income.
The program is slightly inspired by my older project "FastTaxCalc" which I made using Java some time ago.
Software is currently only avaliable in Polish language.

## Features

- Calculating the tax from Polish PIT progressive rates from 1992 to 2025
- Calculating Polish PIT flat tax or CIT including reduced rates and rates dating back to 2000
- Information about current and former tax systems and rates
- Summary of income tax, healthcare bill and solidarity tax when current tax system is selected.
- Export to .txt file option

## Tech used

Program was written in C# programming language with Windows Forms class library using Visual Studio 2022. Icons by [Icon8](https://icons8.com/)
